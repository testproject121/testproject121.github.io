DuckDuckGo Android JS Bridge UXSS - Instant Demonstration

✅ Did you see an alert box?

If YES: The exploit worked! This proves the JavaScript injection vulnerability.

What happened:

  1. Malicious callback registered: alert(document.domain)//
  2. Vulnerable code executed: window['alert(document.domain)//'](...)
  3. JavaScript parsed it as: alert(document.domain) followed by //... (comment)
  4. Alert box displayed = XSS successful!
